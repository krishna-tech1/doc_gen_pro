@router.post("/generate-circular")
def create_circular(payload: CircularRequest, db: Session = Depends(get_db)):
    # Generate AI content
    ai_content = generate_content("circular", context_dict)
    
    # Use template to generate document
    doc_path = process_circular_template(
        title=payload.title,
        date=payload.date,
        time=payload.time,
        venue=payload.venue,
        department=payload.department,
        chief_guest=payload.chief_guest,
        ai_content=ai_content
    )
    
    # Return download
    filename = os.path.basename(doc_path)
    return {
        'message': 'Document generated',
        'preview': ai_content,
        'download_url': f'/generated/{filename}'
    }

========================
TEMPLATE CUSTOMIZATION
========================

To modify the template:

1. Edit backend/templates/create_document_template.py
2. Change logo, fonts, colors, sections as needed
3. Run: python backend/setup_templates.py (regenerates template)
4. Template updates automatically used by processors

Common Customizations:

CHANGE LOGO:
   Pass different logo.png to create_template()

CHANGE FONTS:
   Update: run.font.name = 'Arial'

CHANGE COLORS:
   Update HEADER_COLOR = RGBColor(0x00, 0x00, 0x00)

ADD SECTIONS:
   Add new lines in create_document_template.py with {{NEW_PLACEHOLDER}}
   Update context dict in template_processor.py
